# 足し算れんしゅうアプリ 🎯

子供向けの算数学習Webアプリです。簡単な足し算を楽しく学べます。

## 機能

- **ランダム問題生成**: 0〜10の数字を使った足し算の問題がランダムに出題されます
- **即座のフィードバック**: 正解・不正解を分かりやすく表示
- **スコア記録**: 正解数と総問題数を記録
- **子供向けデザイン**: 明るくカラフルで楽しいUI
- **効果音**: 正解時に楽しい音が鳴ります
- **レスポンシブ対応**: スマートフォンでも快適に使えます

## 使い方

1. **ブラウザで開く**
   - `index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップします

2. **問題を解く**
   - 画面に表示された足し算の問題を解きます
   - 例: 「3 + 4 = ?」なら、答えは「7」です

3. **答えを入力**
   - 答えの数字を入力欄に入力します
   - 「こたえる」ボタンをクリックするか、Enterキーを押します

4. **結果を確認**
   - 正解なら「🎉 せいかい！」と表示されます
   - 不正解なら正しい答えが表示されます

5. **次の問題へ**
   - 「つぎのもんだい →」ボタンをクリックして次へ進みます

## 学習内容

### 足し算とは？

足し算は、2つ以上の数を合わせて合計を求める算数の基本操作です：

- 0 + 1 = 1
- 1 + 2 = 3
- 2 + 3 = 5
- 3 + 4 = 7
- 4 + 5 = 9
- 5 + 6 = 11
- 6 + 7 = 13
- 7 + 8 = 15
- 8 + 9 = 17
- 9 + 10 = 19

### なぜ重要？

足し算は算数の基礎として非常に重要です：
- 繰り上がりのある計算の理解につながります
- 暗算能力の向上に役立ちます
- 数の感覚を養います

## カスタマイズ

### 問題の難易度を変更する

`script.js` の以下の部分を変更できます：

```javascript
// 現在: 0から10までの数字
currentNumber1 = Math.floor(Math.random() * 11);
currentNumber2 = Math.floor(Math.random() * 11);

// 例: 0から5までにする場合
currentNumber1 = Math.floor(Math.random() * 6);
currentNumber2 = Math.floor(Math.random() * 6);
```

### デザインの変更

`styles.css` でデザインをカスタマイズできます：
- 色の変更
- フォントサイズの調整
- レイアウトの変更
